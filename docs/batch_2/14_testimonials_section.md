# Testimonials Section

## Purpose
Reinforce trust through patient experiences.

## Content Structure
- Quote text
- Patient name (first name + initial)
- Optional condition or outcome

## Rules
- Authentic tone
- No clinical promises
- Short to medium length testimonials

## Layout Options
- Horizontal scroll on mobile
- Grid layout on desktop

## Visual Style
- Card-based layout
- Soft shadows or borders
- Calm, reassuring design
