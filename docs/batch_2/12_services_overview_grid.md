# Services Overview Grid

## Purpose
Help users quickly understand available services and navigate deeper.

## Structure
- Grid of service cards (2–4 columns depending on screen size)
- Each card includes:
    - Icon
    - Service name
    - One-line description
    - 'Learn more' link

## Services to Highlight
- Physiotherapy
- Chiropractic Care
- Sports Rehabilitation
- Exercise Physiology
- Workplace Health
- Home Visit Physiotherapy

## Interaction
- Hover state on desktop
- Entire card clickable
- Clear focus state for accessibility

## SEO Notes
- Each service links to a dedicated service page
- Use keyword-rich but natural descriptions
