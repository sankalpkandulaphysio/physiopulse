# Location Page Template

## Goal
Capture local search traffic and nearby intent.

## Structure
1. Hero (Location-specific)
2. Clinic Introduction
3. Services Available at This Location
4. Practitioner / Team Overview
5. Testimonials from Local Patients
6. Contact & Booking Form
7. CTA Banner

## Local SEO Rules
- Include Hyderabad area name in H1
- Add address and contact info
- Optimise meta title and description

## CTA Behaviour
- WhatsApp click-to-chat
- Local phone number if available
