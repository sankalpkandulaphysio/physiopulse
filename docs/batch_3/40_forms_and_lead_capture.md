# Forms & Lead Capture

## Purpose
Collect enquiries and booking requests without payment.

## Form Types
- Appointment request
- Callback request
- General enquiry

## Required Fields
- Name
- Phone number
- Email (optional)
- Preferred service
- Message

## Behaviour
- Simple validation
- Thank-you confirmation message
- Optional WhatsApp redirect

## Privacy
- Clear privacy notice
- No data sharing claims
