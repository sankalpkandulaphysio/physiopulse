# Design System

## Color Palette
- Primary Blue: #00468C
- Accent Blue: #2F80ED
- Light Background: #F7F9FC
- Text Dark: #1F2933

## Typography
- Headings: Sans-serif (e.g., Inter, Open Sans)
- Body: Clean sans-serif, readable on mobile

## Font Scale
- H1: 48px desktop / 34px mobile
- H2: 36px / 28px
- Body: 16px / 15px

## Buttons
- Primary: Solid blue, white text
- Secondary: Outline style

## Spacing
- Section padding: 80px desktop / 48px mobile
- Grid gap: 24px

## Iconography
- Simple line icons
- Medical / movement related
